# Histopatologisk-kreftdeteksjon-med-Convolutional-Neural-Networks-Kaggle

Dette prosjektet implementerer en Convolutional Neural Network (CNN) for å klassifisere metastatisk kreft i histopatologiske bildepatcher fra Kaggle-konkurransen Histopathologic Cancer Detection.

Prosjektet ble utviklet som en del av en universitetseksamen (karakter: A) og demonstrerer en komplett maskinlæringspipeline fra databehandling til modellvalidering og tolkning.
